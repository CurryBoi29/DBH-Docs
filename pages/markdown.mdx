# Sample Markdown

## Links

```mdx
[... Text](... URL)
```

## Images

```mdx
![... Alt Text](... Image)
```

## Code Blocks

Remove the backticks to use the code block

````mdx
```js
console.log('Hello, World!')
``\`
```
````

## Inline Code

```mdx
`console.log('Hello, World!')`
```

## Dropdowns

Dropdown with items:

```mdx
<Dropdown title="..." icon="...">
  <DropdownItem>...</DropdownItem>
  <DropdownItem>...</DropdownItem>
</Dropdown>
```

Dropdown with regular text:

```mdx
<Dropdown title="..." icon="...">
  This is a dropdown
</Dropdown>
```

## HTTP Request

Allowed Options:

- `GET`
- `POST`
- `PUT`
- `DELETE`
- `PATCH`

```mdx
<HTTP method="..." url="..." note?={note} />
```

## Tables

```mdx
| Header 1 | Header 2 | Header 3 |
| -------- | -------- | -------- |
| Cell 1   | Cell 2   | Cell 3   |
| Cell 4   | Cell 5   | Cell 6   |
```

## Lists

Unordered List:

```mdx
- Item 1
- Item 2
- Item 3
```

Ordered List:

```mdx
1. Item 1
2. Item 2
3. Item 3
```

You can also nest lists:

```mdx
- Item 1
  - Item 1.1
  - Item 1.2
- Item 2
- Item 3
```

## Regular Markdown Elements

Regular markdown elements like, headings, paragraphs, blockquotes, bold, italic, etc. are supported.

```mdx
# Heading 1

## Heading 2

### Heading 3

This is a paragraph

> This is a blockquote
> **Bold Text** > _Italic Text_
```
